Flicking
===============
플리킹 컴포넌트.
모바일에서 화면의 플리킹을 제공한다.

## Feature
* 모바일 수직/수평 스와이프
* 순환/비순환 플리킹 제공
* 플리킹시 좌/우 데이터 로드기능
* 사용자 이벤트 제공
* 마그네틱 기능 설정
